4.1 Escopo tecnológico

Agro42, é um robô web com um microcomputador Arduino, que conectado a uma balança fará a pesagem do alimento e toda a automação dos processos, como por exemplo: informar o peso, código de rastreio, informações do produtor e local de origem, emissão de relatórios e de certificados fitossanitários, entre outros processos.

4.1.1 Hardware

O mini mais utilizado no mundo é RaspBarry 3 é a escolha ideal para Agro42 que exige uma quantidade maior de armazenamento melhor conexão de internet.

Porém, não é necessário estar conectado a internet 24 horas por dia para utilizar o equipamento, pois o Raspbarry 3 permite o agricultor trabalhar normalmente sem internet, imprimindo as etiquetas armazenando os dados no banco local o Mongo.js, podendo ser acessado localmente pela tela Touch LCD, por celular ou computador. Para a integração dos dados com o sistema o conjunto conta com ligação à internet via porta LAN, WiFi 802.11n e Bluetooth 4.1 ou 3G(Roteado do celular).

4.1.2 Características técnicas
  • 1 x Raspberry Pi 3 Modelo B Placa
  • 1 x cabo HDMI
  • 1 x USB leitor de cartão TF (Nós enviaremos a cor de forma aleatória)
  • 1x16 GB TF Cartão
  • 1 x Cabo De Alimentação com Interruptor
  • 1 x EUA Plugue De Alimentação 5V3A
  • 1x3.5 " LCD Touch Screen Display
  • 1 x Caixa de Acrilico para proteção

4.1.3 Especificações técnicas

  • 1.2 GHz Quad-Core ARM Cortex-A53
  • Wi-Fi 802.11 b/g/n (2.4GHz), LAN e Bluetooth 4.1 (BCM43438)
  • 802.11 bgn Wireless LAN e Bluetooth 4.1 (Bluetooth Clássico e LE)
  • 1 GB de RAM
  • 1x3.5 " LCD Touch Screen Display
  • CPU de 64 Bits
  • 4 pólos Estéreo de saída e porta de vídeo Composto
  • HDMI em tamanho real
  • 10/100 BaseT Ethernet socketbr
  • DSI exibição porta para conectar a tela sensível ao toque Raspberry Pi
4.1.4 Especificações do Software
